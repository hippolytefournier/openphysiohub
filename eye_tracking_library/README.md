# eye_tracking_library

This directory must contain SR Research proprietary files that **cannot be
redistributed**. Obtain them by installing the **EyeLink Developers Kit**
from the SR Research support portal:

> https://www.sr-support.com/ → Downloads → EyeLink Developers Kit

After installation, copy the following files here:

| File | Source path (typical) |
|---|---|
| `CalibrationGraphicsPygame.py` | `<EyeLink SDK>\SampleExperiments\Python\` |
| `type.wav` | same directory |
| `qbeep.wav` | same directory |
| `error.wav` | same directory |
