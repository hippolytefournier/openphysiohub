# biopac_library

`mpydev.py` is a BIOPAC `BioPac` wrapper class from the **PyGaze** project,
licensed under **GPL v3**.

Source: https://github.com/esdalmaijer/PyGaze

It is **not used at runtime** by OpenPhysioHub (the package talks to
`mpdev.dll` directly via ctypes). It is kept here as a reference
implementation.
